## üöÄ Getting Started

### Prerequisites
- **Java 17+** (OpenJDK or Oracle JDK)
- **PostgreSQL 12+** (running locally or remote)
- **Gradle 8+** (wrapper included: `./gradlew`)
- **Git** (for version control)

### Initial Setup

```bash
# 1. Clone repository
git clone https://github.com/sthwalo/acc.git
cd FIN

# 2. Set up environment variables
cp .env.example .env
# Edit .env with your database credentials:
#   DATABASE_URL=jdbc:postgresql://localhost:5432/drimacc_db
#   DATABASE_USER=your_username
#   DATABASE_PASSWORD=REPLACE_WITH_SECURE_PASSWORD

# 3. Build application
./gradlew clean build

# 4. Run application (choose one mode)
./run.sh                           # Console application (interactive)
java -jar app/build/libs/app.jar api         # API server (http://localhost:8080/api/v1/)
java -jar app/build/libs/app.jar --batch process-transactions  # Batch mode
```

### First-Time Configuration

```bash
# Run the application in console mode
./run.sh

# Follow the setup wizard:
# 1. Create your company
# 2. Initialize Chart of Accounts (76 SARS-compliant accounts)
# 3. Set up fiscal period
# 4. Upload first bank statement (PDF)
# 5. Classify transactions interactively
```

### Quick Test (5 minutes)

See **[QUICK_TEST_GUIDE.md](QUICK_TEST_GUIDE.md)** for detailed testing instructions.

```bash
# Verify database connection
psql -U your_username -d drimacc_db -c "SELECT COUNT(*) FROM companies;"

# Check application status
./gradlew test --tests "ApplicationContextTest"

# Generate test report
./run.sh
# ‚Üí Reports ‚Üí Generate Trial Balance
```
